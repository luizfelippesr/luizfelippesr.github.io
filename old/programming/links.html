Planilhas e afins

Frequentemente, utilizamos planilhas do MS Excel ou LivreOffice Calc para organizar coisas no nosso dia a dia. É possivel ler e modificar estes arquivos facilmente utilizando bibliotecas do Python. Alguns links a respeito

  • https://automatetheboringstuff.com/chapter12/
  • http://openpyxl.readthedocs.io/en/default/

Alternativamente, pode-se querer interagir com as planilhas do Google Sheets. Isso pode ser feito utilizando a API disponibilizada pelo Google:

  • https://developers.google.com/sheets/quickstart/python

Extraindo dados da web

Há um mundo de dados dispersos em páginas web. Pequenos scripts pode ser escritos para visitar um grande número de páginas e extrair o que nos interessa. Isso é particularmente interessante para profissionais da comunicação.

Para executar este tipo de tarefa é necessário empregar módulos como o requests, que permite o conteúdo de páginas seja baixado, e BeautifulSoup, que interpreta elementos html. Alguns exemplos deste tipo de aplicação podem ser vistos em

  • http://journalistsresource.org/tip-sheets/research/python-scrape-website-data-criminal-justice
  • http://programminghistorian.org/lessons/intro-to-beautiful-soup
  • https://automatetheboringstuff.com/chapter11/

Usar outra linguagem de programação

Aprender uma linguagem de programação nova que é necessária ou conveniente para uma dada tarefa específica soa intimidador, mas não precisa ser. Os conceitos básicos de programação e o modo de raciocinar são os mesmos em quaisquer linguagens. Geralmente, basta uma olhada em bom manual ou tutorial para se conseguir completar tarefas simples em uma outra linguagem.

Uma boa maneira de se aproximar de uma linguagem nova é começar olhando códigos escritos na linguagem que conhecemos e ver como a mesma coisa é implementada na outra que queremos aprender. Existe um website dedicado a isso chamado Rosetta Code. Lá é possível comparar códigos para uma mesma tarefa em diferentes (por exemplo, veja como escrever um laço for em 157 linguagens diferentes).
